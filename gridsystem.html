Two-Column Layout Example

Using Floats and CSS Grid

Float-Based Left Column

This section is aligned to the left using the float: left; property.

Floats were commonly used before modern layout systems like Flexbox and Grid became standard.

Float-Based Right Column

This section is aligned to the right using float: right;.

It contains secondary information like ads, links, or a sidebar.

Grid Left Column

This section uses the CSS Grid system with grid-template-columns to define two columns.

Grid layout provides better control over spacing, alignment, and responsiveness.

Grid Right Column

This is the second column in the grid layout. It adjusts automatically based on screen size.
